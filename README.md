# BMI Calculator - BMI 計算器
### 功能
* 判斷體重的範圍，正常、過輕、過重。
* BMI 四捨五入到小數點後第一位。

| 成人肥胖定義 |                                    身體質量指數(BMI)(kg/m^2)                                    |
|:------------:|:-----------------------------------------------------------------------------------------------:|
|   體重過輕   | BMI<18.5                                                                                        |
|   健康體位   | 18.5<=BMI<24                                                                                    |
|   體位異常   | 過重：24<=BMI<27<br>輕度肥胖：27 <= BMI < 30<br>中度肥胖：30 <= BMI < 35<br>重度肥胖：BMI >= 35 |

```
身高(cm)：180
體重(kg)：60

您的BMI：18.5
您的體重正常！
```
